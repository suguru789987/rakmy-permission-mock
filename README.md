# ラクミー 経営管理｜権限設定 モック

会社画面の権限設定UI（権限一覧＋権限作成・編集）の動作モック。

公開デモ: https://suguru789987.github.io/rakmy-permission-mock/

## 権限一覧マトリクス

ロール×権限カテゴリの一覧。カテゴリは2群（**閲覧権限**＝分析・レポート・管理画面の閲覧 ／ **操作・設定権限**）に分類。

### 列表示プルダウン（各列ヘッダ内）
- 列ヘッダの **閲覧権限 ▼ ／ 操作・設定権限 ▼** から表示カテゴリ列を選択（パネルは表外にフロート表示）。
- パネル上部に **「全て」**（チェックで全カテゴリ列を展開）、続けて各カテゴリを複数選択。
- 個別カテゴリを選ぶと「全て」は自動解除。フッターは **クリア（全解除＝内訳に戻る）／閉じる**。
- 既定（未選択）は各群1列の **内訳** 表示。

### 内訳セル（列名に沿った許可数）
折りたたみ時、ロールごとに「許可できるカテゴリ数 / 総数」を列名に沿って表示。
- **閲覧権限 列** … `閲覧 N/総数`（閲覧できるカテゴリ数＝操作可＋閲覧のみ）
- **操作・設定権限 列** … `操作 N/総数`（操作できるカテゴリ数）

操作数が多いほど権限の強い（＝変更権限の大きい）ロール。

## 権限設定（カテゴリ詳細）

- 機能カテゴリタブで設定一覧を切替。
- 機能の性質ごとに状態を可変化（閲覧専用＝なし/閲覧、設定＝なし/閲覧/操作、実行・管理＝なし/可、常時＝閲覧固定）。
- 新規実装分は ★★ バッジ。

## ロール構成

- **オーナー（最上位・全権）** … 全権固定（編集不可）。全権の付与先を指名可。
- **会社ロール（全社・本部＝職務分掌）** … 経理担当・人事担当など。操作・設定権限を職務に応じて付与。
- **店舗ロール（店舗現場）** … 自店舗スコープ。閲覧中心、操作は限定。

スコープ規則：店舗スコープの「従業員」は自店舗のみ／全社集計指標など会社スコープ専用は店舗ロールに付与不可。
