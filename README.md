# ラクミー 経営管理｜権限設定 モック

会社画面の権限設定UI（権限一覧／権限編集／指標カタログ／役割サマリ／ユーザー割当）の動作モック。

公開デモ: https://suguru789987.github.io/rakmy-permission-mock/

## 画面構成

権限一覧の上部ナビは用途で2セットに分離:
- **権限の設定** … `権限編集` ／ `指標カタログ`
- **割当・俯瞰** … `役割サマリ` ／ `ユーザー割当`

## 権限一覧マトリクス

ロール × 権限カテゴリの一覧。**一覧画面権限 / 設定画面権限** の2タブで出し分け（各タブはその群のカテゴリのみ表示）。

- **一覧画面権限**（青）＝ 一覧・分析画面の閲覧可否
- **設定画面権限**（緑）＝ 設定画面の操作可否

左端は **区分列**（オーナー／会社／店舗）を行にまたがり表示。権限名・操作列は固定幅でロックし、カテゴリ列が増えると横スクロール。

### セル表示
- **一覧画面**：`閲覧`（見られる）／`一部`（一部のみ）／`—`（なし）
- **設定画面**：`操作`＋`閲覧可`（操作は閲覧を内包）／`閲覧可`（見るだけ）／`—`（なし）
- **「一部」** ＝ カテゴリ内の複数機能で設定が混在している状態。
- **セルをクリック** → そのロール×カテゴリの設定詳細へ直接遷移（オーナーは固定のため対象外）。
- 折りたたみ時の内訳セルは列名に沿った許可数（一覧＝`閲覧 N/総数`、設定＝`操作 N/総数`）。

### 列の表示（ツールバー）
- **列プルダウン**（表外固定・横スクロールで動かない）：`全て`＋各カテゴリを複数選択。クリア／閉じる、未選択は内訳1列。
- **💾 保存** … 現在の表示列に名前を付けて保存（アプリ内モーダル）。
- **📑 保存ビュー** … 保存した表示列を1クリックで適用／削除。
- 列選択は `localStorage` に自動記憶 → 次回アクセス時も同じ列。

## 権限編集

- **一覧画面権限 / 設定画面権限** の2タブ。各タブ最上部に **全カテゴリ一括設定（初期値）** バー（一覧＝閲覧/なし、設定＝全て/操作/閲覧/なし）。
- カテゴリ単位で一括、「詳細 ＞」で機能・操作・指標を個別設定。混在は「一部」表示。
- 機能の性質ごとに状態可変（閲覧専用＝なし/閲覧、設定＝なし/閲覧/操作/全て、実行・管理＝なし/可、常時＝閲覧固定）。新規実装は ★★ バッジ。

## ロール構成

- **オーナー** … 全権固定（編集不可）。全権の付与先を指名可。
- **会社ロール（全社・本部＝職務分掌）** … 経理担当・人事担当など。設定権限を職務に応じて付与。
- **店舗ロール（店舗現場）** … 店長・従業員・エリア長。自店舗スコープ、閲覧中心。

スコープ規則：店舗スコープの「従業員」は自店舗のみ／全社集計指標など会社スコープ専用は店舗ロールに付与不可。

## ユーザー割当

- ユーザーに権限ロールを割当（複数可）。複数ロールは権限が合算される。
- **有効範囲** 列＝スコープ（自店舗／全社／指定店舗群）を表示。未割当は「なし」。

## 権限の階層

1機能の中では `なし ⊂ 閲覧 ⊂ 操作 ⊂ 全て` の順序（操作は閲覧を内包）。表示専用画面と操作画面は別カテゴリとして分離（例：`従業員（閲覧）` と `従業員管理`）。
